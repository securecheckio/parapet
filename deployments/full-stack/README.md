# Full-stack Deployments

Use this option when you need the complete operational stack: proxy, API, Redis, and dashboard.

## Includes

- Parapet RPC proxy
- Parapet API for dynamic rule and escalation workflows
- Redis shared state
- Dashboard for monitoring and approvals

## Deployment Methods

- [`docker-compose/`](./docker-compose/) - Full stack via Docker Compose
- [`retrofit/`](./retrofit/) - Advanced dashboard/API retrofit deployment

## Best For

- Teams needing centralized monitoring
- Human-in-the-loop approval workflows
- Multi-operator or managed environments
